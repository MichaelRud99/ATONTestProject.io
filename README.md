### Built With

The project is built on a JS-library for creating a user interface React using libraries:

-  react-redux
-  react-router
-  redux-saga
-  react-transition-group
-  normalizr
-  uniqid
-  sass

## How To Use

For correct use, you need to install the following dependencies

1. npm ci
2. npm run start
